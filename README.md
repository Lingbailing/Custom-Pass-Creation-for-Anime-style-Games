# Custom-Pass-Creation-for-Anime-style-Games
这是一个基于epyuri开发的项目二次开发的动漫风格游戏自定义通行证。它允许你自定义和创建通行证
原项目地址----https://github.com/epYuriX/mrfztxz 
我也不懂其中功能好不好用
## 实时预览: 支持双画布前/背面实时渲染（canvas01 / canvas02），编辑后即时重绘预览。
## 模板驱动渲染: 采用可解析的模板层结构（group/image/text/color 等），渲染逻辑集中在解析器，易扩展自定义模板。
## 强大的文本定制: 高级面板可实时编辑文本内容、位置、旋转、字间距、水平缩放、字体/字重、字号与颜色，并支持上传自定义字体。
## 可编辑人物图像: 支持上传角色图、平移拖拽和鼠标滚轮缩放（保存运行时状态），并自动同步背面镜像渲染与着色。
## 自定义图标与阵营/职业: 支持职业/阵营选择与上传自定义图标，灵活替换模板素材。
## 导出便捷: 一键导出正面/背面 PNG（通过 canvas.toDataURL + 下载链接）。
## 视觉效果控制: 渐变、挡板、背影颜色、Logo 与切割线等可见性与颜色均可配置。
## 字体加载与高质量排版: 使用 FontFace 与 document.fonts 动态加载字体，含 pt→px 转换以支持更准确的字号渲染。
## 容错与运行时覆盖: 通过 runtimeImageMap/runtimeLayerState 实现运行时资源覆盖与状态保存，支持用户自定义资源替换且兼容性好。

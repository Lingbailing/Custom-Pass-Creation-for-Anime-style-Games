# 自定义动漫风格通行证创建工具

该项目是由Lingbailing二次开发的自定义通行证创建工具。原作者epYuriX,原仓库：https://github.com/epYuriX/mrfztxz

## 概述
该工具允许用户使用各种动漫风格元素创建和自定义游戏通行证，提供友好的用户界面用于设计和导出。

## 功能
- **通行证创建**：设计自定义通行证，包括多层背景、角色和装饰元素。
- **字体选择**：从多种字体中选择，以匹配动漫风格。
- **图像导入和处理**：直接在工具内导入、调整大小和编辑图像。
- **导出选项**：导出通行证为多种格式，如PNG、JPEG和SVG，用于不同用途。
- **用户友好界面**：直观的控制用于轻松导航和自定义。

## 使用方法
1. 在代码编辑器或IDE中打开项目。
2. 打开https://lingbailing.github.io/Custom-Pass-Creation-for-Anime-style-Games/体验功能。
